# JupyterLab as Jupyter Server Extension

DEPRECATED [WIP] Jupyter server [jupyterlab#4954](https://github.com/jupyterlab/jupyterlab/pull/4954)

```bash
jupyter server --ServerApp.jpserver_extensions="{'jupyterlab': True}"
jupyter server --ServerApp.jpserver_extensions="{'jupyterlab': True, 'notebook': True}"
```
