# JupyterLab as Jupyter Server Extension

+ [JupyterLab as Server Extension](https://github.com/jupyterlab/jupyterlab/pull/7416)
+ [JupyterLab Server as Server Extension](https://github.com/jupyterlab/jupyterlab_server/pull/79)

```bash
jupyter server --ServerApp.jpserver_extensions="{'jupyterlab': True}"
jupyter server --ServerApp.jpserver_extensions="{'jupyterlab': True, 'notebook': True}"
```
