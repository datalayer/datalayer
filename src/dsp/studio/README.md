# Datalayer DSP Studio

:chart_with_upwards_trend: :bar_chart: Datalayer DSP Studio is an open source tool to discover, collaborate and analyze data.

This folder contains the `Datalayer Studio` source code that allows accessing the `Datalayer Science Platform` in a visual way.

The Studio will help you create and manage your `K8S` cluster and transform it into a powerful `JupyterLab` solution for your Big Data Science projects to create

👉 exercise  
👉 cards  
👉 timeline  
👉 paper  

```bash
# Print the help.
cd $DLAHOME/src/dsp/studio && \
  make help
```
