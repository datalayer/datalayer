# Datalayer Explorer

:chart_with_upwards_trend: :bar_chart: Datalayer Explorer is an open source tool to discover, collaborate and analyze data.

This folder contains the `Datalayer Explorer` source code that allows accessing the `Datalayer Science Platform` in a visual way.

The Explorer will help you create and manage your `K8S` cluster and transform it into a powerful `JupyterLab` solution for your Big Data Science projects.

```bash
# Print the help.
cd $DLAHOME/src/dsp/explorer && \
  make help
```
