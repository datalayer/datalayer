## VDOM with @jupyterlab/vdom

JupyterLab [VDOM](https://github.com/jupyterlab/jupyterlab/tree/master/packages/vdom) and [VDOM Extension ](https://github.com/jupyterlab/jupyterlab/tree/master/packages/vdom-extension) packages which use the described `@nteract/transform-vdom` package.

- [Add event handling support to vdom-extension](https://github.com/jupyterlab/jupyterlab/pull/5670).
- [Nteract Python VDOM](https://github.com/nteract/vdom) is Virtual DOM for Python.
