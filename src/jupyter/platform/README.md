# 🌕 Jupyter Platform

> Jupyter Platform, a Jupyter Distribution for Academics and Business.

This repository bundles selected projects of the large [Jupyter](https://jupyter.org) ecosystem. Those projects are chosen to create a useful platform for Academics and Business to analyse and share data analysis.

A projects is implemented by a well defined set of repositories and offer a well defined functionality to the end-user. Those repositories are maintained under the Jupyter umbrella, by other third-party partners or by Datalayer it-self.

## RJK Stack

Datalayer provides the missing pieces to create React-Jupyter-Kubernetes (a.k.a. `RJK`) stack.

## Learn by Example

We have curated useful [examples](https://github.com/datalayer/jupyter-platform/tree/main/examples) to allow a Developer shim into the platform.
