# 🌕 Jupyter Platform Examples

Useful examples to develop on top of the Jupyter Platform.

```bash
# Create your conda environment.
conda deactivate && \
  conda remove -y --all -n jupyter-platform
conda env create -f environment.yml
conda activate jupyter-platform
```
