# JupyterLab Test

```bash
# Run all tests.
cd $DLAHOME/repos/jupyterlab && \
  yarn build:test && \
  yarn test
```

```bash
# Run only specific tests.
cd $DLAHOME/repos/jupyterlab/tests/test-services && \
  yarn run build && \
  yarn run test
```
