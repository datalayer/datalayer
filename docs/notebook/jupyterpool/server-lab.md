# Jupyter Lab Server

```
BROWSER --- LAB_SERVER --- NOTEBOOK_SERVER --- KERNEL
```

JupyterLab Server [GitHub](https://github.com/jupyterlab/jupyterlab_server) repository.

The purpose of `jupyterlab_server` is to enable custom spins of JupyterLab, like apps that don't need the frontend JS from lab itself. An example of this is [voila](./../msc/voila.md).

## Start JupyterLab with Notebook Server

```bash
jupyter lab
```
