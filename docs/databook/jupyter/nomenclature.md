# Jupyter Nomenclature

Common vocabulary and concept are needed.

## Notebook vs Note

A `Notebook` should be named a `Note`.

## Kernel vs Server

A `Server` should be named a `Kernel`.

## Client

A `Client` should be named a `Kernel Client`.

## Authentication Token

`Authentication Token` machinery should be pluggable.
