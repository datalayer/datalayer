# Papermill

> [Papermill](https://github.com/nteract/papermill) is a tool for parameterizing, executing, and analyzing Jupyter Notebooks.

Papermill [Docs](https://papermill.readthedocs.io/en/latest).

[R Binding](https://github.com/nteract/papermillr).

## Piping

```bash
cat in.ipynb | papermill > out.ipynb
```
