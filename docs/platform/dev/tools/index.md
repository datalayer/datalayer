## JSON

Nicely indented JSON is very easy to understand.

If you get a large piece of non-indented JSON somehow, and are trying to make sense of it, you can cut and paste into one of the online validators.

+ http://jsonlint.com
+ http://jsonformatter.curiousconcept.com

Both of these validators will indent your JSON, even when it contains extensions unsupported by them (such as comments or bare strings).
