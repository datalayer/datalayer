# Datalayer Value

For `Data Scientist` and `Business` stakeholders.

+ [Data Storytelling](/about/value/data-storytelling.md). Ship beautiful User Interface for Data Scientists and Business.
+ [Education](/about/value/education.md).
+ [Open Science](/about/value/open-science.md).
+ [Open Data](/about/value/open-data.md).
+ [Collaborative Discovery](/about/value/collaborative-discovery.md).

For `Devops` stakeholders.

+ [Cost-Less](/about/value/cost-less.md) with cheap resources your can schedule on any cloud.
+ [IT-Less](/about/value/it-less.md). You can setup Big Data in the Cloud as easily as installing a software on your laptop.
+ [Serverless](/about/value/serverless.md). Don't bother with Virtual Machines anymore.
+ [No Vendor nor Cloud Lock-In](/about/value/no-lock-in.md). Feel free to move whenever you want and reuse 100% of your code and science investements.
+ [Exploration to Alpha](/about/value/exploration-to-alpha.md) to create scalable Data Products.
