# Studio

`Datalayer Studio` is the collaborative Data Science workbech to analyse your datasets and insights.

The Studio can run on a cluster created and managed by [Kuber](/about/inside/kuber.md) or on any other Kubernetes compliant cluster such as [Amazon EKS](https://aws.amazon.com/eks), [Azure AKS](https://azure.microsoft.com/en-us/services/container-service)...
